# Edit Distance Visualizer

Interactive React visualization of the Levenshtein (Edit Distance) dynamic programming algorithm.

## Features

- Step-by-step DP table filling
- Shows ALL tried operations (insert / delete / match / replace)
- Highlights chosen operation
- Optional optimal backtrace
- Playback controls

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
